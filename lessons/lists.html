What is a list?

A list is an ordered, mutable collection of items. It can hold mixed types and supports indexing, slicing, and common operations like append, insert, and remove.

Creating lists

# Empty list
lst = []

# With elements
lst = [1, 2, 3, 4, 5]
lst = ["a", "b", "c"]
lst = [1, "hello", 3.14, True]  # mixed types

Access and slicing

lst = [10, 20, 30, 40, 50]
print(lst[0])    # 10
print(lst[-1])   # 50
print(lst[1:4])  # [20, 30, 40]
