What is an algorithm?

An algorithm is a finite sequence of well-defined steps that takes some input and produces an output. It should be unambiguous, correct, and efficient.

Pseudocode

Pseudocode is a high-level description of logic without language-specific syntax. It helps plan before coding.

// Example: find max in array
MAX = A[0]
FOR i FROM 1 TO length(A) - 1
    IF A[i] > MAX THEN
        MAX = A[i]
RETURN MAX
