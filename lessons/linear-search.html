Linear Search

Searching — scan until found

0% course complete

Idea

Linear search checks each element one by one until the target is found or the list ends. Works on any order of elements.

Algorithm

Traverse the array from start to end. If arr[i] == target, return i. If the loop ends, return -1 (not found).

Python code

def linear_search(arr, target):
    for i in range(len(arr)):
        if arr[i] == target:
            return i
    return -1

arr = [10, 25, 30, 45, 50]
print(linear_search(arr, 30))   # 2
print(linear_search(arr, 99))   # -1
