Why complexity matters

Time complexity describes how runtime grows with input size; space complexity describes extra memory used. We use Big-O notation to express worst-case growth.

Common complexities

NotationMeaningExample
O(1)ConstantIndex access, fixed work
O(log n)LogarithmicBinary search
O(n)LinearSingle loop over input
O(n²)QuadraticNested loops over input
